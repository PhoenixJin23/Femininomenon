# Femininomenon

东西方的父权文化，在寻找男人优越，文人等而下之的根据时，不约而同地从直观生理现象入手，竭力渲染“男强女弱”的天然法则，并把这一逻辑无限制地推及到男女两性对比的各个方面。（欧阳洁《女性与社会权力系统》）

如果只是简单认为生理条件更有优势就应该占统治地位，那么当下被崇拜的肤色应该是黑色，有话语权的应该是黑色人种。黑色人种的生理条件明显优于黄色和白色人种，却在人文社会中长期处于弱势地位，这足以证明将父权文化归因为生理条件使然是明显不成立的。


它揭示了一个至关重要的动作：渲染。

关键不在于男女有没有生理差异（差异是客观存在的），而在于社会文化如何选择、放大和解释这些差异。

•选择性关注：父权文化只死死盯住“男性肌肉更发达”这类体能上的差异，却选择性地忽略女性在免疫系统、平均寿命、疼痛耐受度等生理维度上的显著优势。

•无限推及：然后，它做了一个逻辑上的巨大跳跃——把体能差异这个单一维度的生理事实，无限制地推及到“智力、理性、领导力、情绪稳定性”等所有社会评价维度上，构建出一套“男强女弱”的“全方面”叙述。

•倒果为因：这套叙事做得最巧妙的一步是，把长期社会规训的结果（比如女性被排除在公共事务之外，缺乏锻炼某些能力的机会），说成是“天生不适合”的原因。

所以，欧阳洁告诉你，“男强女弱”不是被发现的天然法则，而是被创造出来的文化叙事。



一点延伸思考

当我们用“黑人体魄”这个例子去打穿“男性统治纯属生理使然”时，我们其实也打开了一个更开阔的思考空间：

生理差异从来不是直接产生社会压迫的独立变量，它必须通过特定的社会经济结构才能被转化为权力和等级。就像在采集狩猎社会，女性的采集贡献可能是更稳定的食物来源；到了农业社会，体力需求上升，男性优势才被放大；而到了脑力与协作日益重要的信息时代，体能的决定性作用理应消退。

但父权叙事并不会自动退场。它会不断寻找新的“生理根据”来维护旧秩序。

这就是为什么我们需要持续地去识别这种“渲染”，并且看清：那些被奉为天经地义的等级，在另一个社会、另一个时代，可能根本就不是这个样子。

所以，当任何人再拿“男女生理不同所以社会角色理应不同”说事时，你只需要轻轻地问一句：“那么，按照这个逻辑，为什么人类社会不是由那些体魄最强健的族群来统治的呢？”这便是最好的思考和反诘。
